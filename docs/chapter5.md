# 第五章：财务报表与基本面分析

> 股价是市场给出的价格，财务报表是你判断这个价格是否合理的依据。

---

## 5.1 三张核心报表

每家上市公司每季度、每年都必须公开披露三张财务报表，这是了解一家公司最权威的一手资料。

| 报表 | 回答的问题 | 时间维度 |
|---|---|---|
| **利润表** | 这家公司赚了多少钱？ | 一段时间内的流量 |
| **资产负债表** | 这家公司有多少家底、欠多少债？ | 某一时间点的存量 |
| **现金流量表** | 实际收到和花出去了多少真金白银？ | 一段时间内的流量 |

三张报表相互印证，互为补充。只看一张容易被误导。

---

## 5.2 利润表速读

利润表（Income Statement / P&L）记录一段时间内公司的经营成果。

基本结构：

```
营业收入
- 营业成本
= 毛利润
- 销售费用、管理费用、研发费用
= 营业利润（EBIT）
- 财务费用（利息支出）
+ 投资收益、其他非经营损益
= 税前利润（EBT）
- 所得税
= 净利润
```

**关键指标**：
- **营业收入**：公司卖产品/服务赚了多少钱，是增长的基础。
- **营业成本**：公司卖产品/服务时，使用的成本，是收入减少的原因。
- **毛利润** = 营业收入 - 营业成本：反映产品的盈利能力，毛利润高说明公司在产品上有定价权或成本优势
- **净利润** = 税前利润 - 所得税：反映公司在产品上所赚的利润，净利润高说明公司在产品上有收入优势
- **毛利率** = 毛利润 / 营业收入：反映产品本身的盈利能力，越高说明产品竞争力越强（茅台毛利率 90%+，普通制造业 20%）
- **净利率** = 净利润 / 营业收入：最终的盈利效率，扣除所有费用后每 100 元收入能赚多少钱
- **净利润增速**：同比增长率是判断公司成长性的核心指标

**常见陷阱**：净利润高不一定真的赚钱。如果利润里有大量"投资收益"、"政府补贴"等非经常性损益，可能不可持续，要剔除看扣非净利润。

## 5.3 资产负债表速读

资产负债表（Balance Sheet）是公司某一天的财务快照。

**会计恒等式**：资产 = 负债 + 股东权益

结构：

```
资产（公司拥有的）            负债（公司欠别人的）
  流动资产                      流动负债
    - 货币资金（现金）             - 短期借款
    - 应收账款                    - 应付账款
    - 存货                        - 预收款项
  非流动资产                    非流动负债
    - 固定资产（厂房、设备）        - 长期借款
    - 无形资产（专利、品牌）        - 应付债券
    - 长期股权投资
                                  股东权益
                                    - 注册资本
                                    - 留存收益
```

**关键指标**：

- **货币资金**：现金储备，越充裕抗风险能力越强
- **应收账款**：客户欠公司的钱。应收账款长期大幅增长且占收入比例很高，可能说明收款困难
- **资产负债率** = 总负债 / 总资产：反映公司的杠杆水平。一般制造业 50% 以下相对安全；银行由于业务特性，负债率天然很高（90%+）
- **商誉**：收购其他公司溢价买入的部分。商誉减值是常见的"业绩地雷"

---

## 5.4 现金流量表速读

现金流量表（Cash Flow Statement）是三张报表中最难造假的一张，记录实际的现金进出。

分为三部分：

**经营活动现金流**：日常经营产生的现金，是最核心的部分。
- 持续为正 → 公司真的在赚钱
- 净利润高但经营现金流持续为负 → 警惕，利润可能是"纸面利润"

**投资活动现金流**：买卖设备、收购、投资理财等。
- 扩张期的好公司往往大量资本支出，投资现金流为负，是正常的
- 关注：资本支出是否用于主业扩张，还是乱投资

**融资活动现金流**：借款、还款、分红、增发股票等。
- 频繁增发股票 → 稀释现有股东权益，需警惕
- 稳定分红 → 说明公司现金充裕，对股东友好

**最重要的指标：自由现金流**

```
自由现金流 = 经营活动现金流 - 资本支出（购置固定资产）
```

自由现金流是公司在维持正常运营和扩张之后**真正剩下的可支配现金**，是支撑长期股价的根本。

## 5.5 核心财务指标

把财务报表的数字变成可以横向比较的比率：

### PE（市盈率）= 股价 / 每股收益
含义：你花多少钱买一块钱的利润。PE=20 意味着以当前盈利水平，20 年回本。

- PE 越低，通常越便宜（但也可能是公司出了问题）
- 成长型公司 PE 高（市场在为未来增长付费），成熟型公司 PE 低
- 动态 PE（用未来预测利润）比静态 PE（用过去利润）更有参考价值

### PB（市净率）= 股价 / 每股净资产
含义：你花多少钱买一块钱的账面净资产。

- PB < 1：市场认为公司价值低于其账面净资产（可能是机会，也可能是陷阱）
- 适用于银行、地产等重资产行业；对轻资产的科技公司参考意义有限

### ROE（净资产收益率）= 净利润 / 平均净资产
含义：股东每投入一块钱能赚多少钱，是衡量公司盈利能力最核心的指标之一。

- ROE > 15% 通常被认为是优质公司的标志
- 巴菲特极为看重 ROE：长期持续高 ROE 的公司，往往有强大的护城河

### 负债率 = 总负债 / 总资产
含义：资产中有多少是借来的。负债率过高，财务风险大，利率上升或经济下行时容易出问题。

## 5.6 行业差异：为什么银行 PE 低、科技公司 PE 高

用同一把尺子衡量不同行业是危险的。

**银行**：PE 通常 5–8 倍。不是因为便宜，而是因为：
- 盈利增速慢，资本要求高
- 存在大量表外风险（坏账），利润不完全可信
- 高杠杆行业，风险用 PB 和 ROE 衡量更合理

**科技成长公司**：PE 可能 50–100 倍甚至更高。不是因为贵，而是因为：
- 市场预期未来利润会大幅增长
- 当前利润低但现金流好，或处于战略性亏损扩张期

**周期性行业（钢铁、煤炭、化工）**：PE 在行业景气高峰时往往最低（利润最高），但恰恰是风险最大的时候，因为利润不可持续。买周期股反而要在 PE 高、甚至亏损时买入。

**公用事业（电力、水务）**：PE 中等、PB 中等，以稳定股息为主要回报方式，适合追求稳定收益的投资者。

## 5.7 护城河理论

**护城河（Economic Moat）**来自巴菲特，指一家公司能够长期抵御竞争者侵蚀的可持续竞争优势。

没有护城河的公司，高利润会引来竞争者，利润最终回归平均。有护城河的公司能长期维持高 ROE。

**五种常见护城河类型**：

| 护城河类型 | 说明 | 案例 |
|---|---|---|
| 品牌溢价 | 消费者愿意为品牌多付钱 | 茅台、爱马仕 |
| 网络效应 | 用户越多，产品价值越高 | 微信、Visa |
| 成本优势 | 以更低成本生产同等产品 | 宁德时代（规模）、拼多多（供应链） |
| 转换成本 | 用户替换产品的代价高 | SAP（企业 ERP）、微软 Office |
| 牌照/专利 | 政策或法律壁垒 | 茅台产区、药企专利 |

判断护城河的实用方法：**假设有一家财力雄厚的竞争者模仿这家公司的一切，5 年后能否撼动它的市场地位？**

## 5.8 如何找到财报：公告平台与数据源

**官方一手数据（最权威）**：
- [上交所](http://www.sse.com.cn/)：沪市上市公司公告
- [深交所](http://www.szse.cn/)：深市上市公司公告
- [巨潮资讯](http://www.cninfo.com.cn/)：汇总沪深两市所有公告，是 A 股散户最常用的查公告平台

**数据分析工具**：
- [东方财富 Choice](https://choice.eastmoney.com/)：A 股主流数据终端（收费）
- [Wind](https://www.wind.com.cn/)：机构级别数据终端（收费，昂贵）
- [同花顺 iFinD](https://www.10jqka.com.cn/)：散户常用，部分功能免费
- [雪球](https://xueqiu.com/)：财报数据 + 社区讨论，适合散户

**美股财报**：
- [SEC EDGAR](https://www.sec.gov/edgar/)：美国证监会，所有美股公司的官方申报文件（10-K 年报、10-Q 季报）
- [Macrotrends](https://www.macrotrends.net/)：免费历史财务数据可视化

## 本章小结

| 概念 | 核心要点 |
|---|---|
| 利润表 | 看毛利率、净利率、净利润增速 |
| 资产负债表 | 看现金储备、负债率、商誉风险 |
| 现金流量表 | 经营现金流是否健康，自由现金流是核心 |
| PE/PB/ROE | 跨行业比较需谨慎，各行业估值逻辑不同 |
| 护城河 | 长期持有的前提，判断竞争优势是否持续 |
