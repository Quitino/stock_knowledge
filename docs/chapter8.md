# 第八章：交易策略概览

> 没有放之四海皆准的策略，只有适合自己认知、性格和时间投入的策略。

---

## 8.1 长期投资 vs 短期交易：两种截然不同的思维框架

这不只是持仓时间的区别，而是两种根本不同的盈利逻辑：

| 维度 | 长期投资 | 短期交易 |
|---|---|---|
| 盈利来源 | 企业成长带来的价值增长 + 分红 | 价格短期波动 |
| 核心技能 | 企业分析、行业判断、耐心 | 盘面感知、快速决策、纪律执行 |
| 时间投入 | 相对低（找好公司后等待） | 极高（需要盯盘） |
| 交易频率 | 极低（年均几次） | 极高（日内或数日） |
| 税费影响 | 低 | 高（频繁交易摩擦成本累积） |
| 核心风险 | 公司基本面恶化、估值过高买入 | 预测错误、纪律崩溃 |

对大多数有正式工作的程序员，**长期投资或指数投资**是更现实、更适合的路径——短线交易需要大量时间盯盘，且与职业机构交易者竞争，胜算有限。

## 8.2 价值投资：买公司而非买股票

价值投资由本杰明·格雷厄姆奠基，巴菲特将其发扬光大。

**核心思想**：
- 股票是公司所有权的一部分，不是用来投机的代码
- 市场先生（Mr. Market）是你的仆人，而不是向导——他每天给你报一个价格，你可以选择接受或无视
- 以低于内在价值的价格买入优秀公司，等待市场重新认识其价值

**操作框架**：

1. 寻找具有持续竞争优势（护城河）的公司
2. 估算其内在价值
3. 等待市场给出有安全边际的价格（一般是内在价值的 6–7 折以下）
4. 买入并长期持有，除非基本面恶化或价格大幅高于内在价值

**代表人物**：
- 本杰明·格雷厄姆（Benjamin Graham）：《聪明的投资者》、《证券分析》
- 沃伦·巴菲特（Warren Buffett）：伯克希尔·哈撒韦
- 查理·芒格（Charlie Munger）：多学科思维模型
- 彼得·林奇（Peter Lynch）：《彼得·林奇的成功投资》

**适合的人**：有耐心、不频繁看盘、对企业分析有兴趣的长期主义者。

## 8.3 趋势交易：顺势而为的逻辑

**核心思想**：不预测市场方向，而是在趋势确认后跟随，在趋势破坏时离场。

"截断亏损，让利润奔跑"——趋势交易最核心的原则。

**典型框架**（海龟交易系统的变体）：

1. 价格突破近 N 日高点 → 买入
2. 价格跌破近 M 日低点 → 止损离场
3. 持仓期间设置移动止损，保住已有利润
4. 不预测顶部，等趋势破坏信号出现再离场

**特点**：
- 胜率不高（可能 40% 甚至更低），但靠"大赚小亏"整体盈利
- 需要极强的纪律性，止损时必须严格执行
- 强趋势市场中效果极好，震荡市场连续止损

**代表方法论**：
- 海龟交易法（Richard Dennis / William Eckhardt）
- 趋势跟踪基金（CTA 策略）

## 8.4 量化交易简介：程序员的天然优势

量化交易（Quantitative Trading）用数学模型和计算机程序替代人工判断，系统地执行交易策略。

**程序员做量化的优势**：
- 能写代码，回测框架、数据处理不是障碍
- 习惯数据驱动，不容易被情绪左右
- 能快速迭代验证策略想法

**量化交易的核心流程**：

```
想法 → 数据获取 → 策略编写 → 历史回测 → 参数优化 → 模拟交易 → 实盘验证
```

**几种常见量化策略类型**：

| 策略类型 | 逻辑 | 适合周期 |
|---|---|---|
| 动量策略 | 过去涨得好的继续买 | 中短期 |
| 均值回归 | 偏离均值后会回归 | 短期 |
| 多因子选股 | 用多个因子（PE、ROE、动量…）筛选股票 | 中长期 |
| 统计套利 | 寻找价格相关资产间的偏差 | 短期 |
| 高频交易 | 毫秒级别利用微小价差 | 极短（散户不可参与）|

**入门工具**：
- Python + [akshare](https://github.com/akfamily/akshare)（免费 A 股数据）
- [backtrader](https://www.backtrader.com/)（回测框架）
- [ricequant](https://www.ricequant.com/)（国内量化平台，有回测环境）
- [qlib](https://github.com/microsoft/qlib)（微软开源量化研究框架）

**警告**：回测效果好 ≠ 实盘赚钱。过拟合（Overfitting）是量化策略最常见的陷阱——在历史数据上训练出来的参数，在未来数据上往往失效。

## 8.5 指数投资（被动投资）：战胜大多数主动基金的方法

**核心观点**：大多数主动基金经理长期跑不赢市场指数，定投宽基指数对普通投资者而言是最优解之一。

**数据支持**：
- 标普道琼斯指数公司发布的 SPIVA 报告显示，长达 15–20 年的区间内，超过 90% 的美国主动管理股票基金跑不赢标普 500
- 中国市场主动基金超额收益更不稳定，长期胜率同样有限

**为什么**：
- 市场是所有参与者的平均，所有人加总必然等于市场平均
- 主动基金收取更高的管理费，长期来看费率是拖累
- 优秀基金经理难以持续识别，且优异业绩会吸引大量资金规模限制操作空间

**推荐标的（按风险递增）**：
- 货币基金 → 债券 ETF → 沪深 300 ETF → 中证 500 ETF → 纳斯达克 100 ETF

**定投策略**：每月固定日期买入固定金额，不择时，长期坚持。

- 优点：纪律性强、平滑成本、无需分析能力
- 缺点：在长期熊市中体验极差，需要心理素质

## 8.6 网格交易：区间震荡时的机械策略

**适用场景**：判断某资产未来一段时间会在一个区间内震荡，上不去也下不来。

**核心逻辑**：
1. 设定一个价格区间（如 10–20 元）
2. 将区间均匀切割成 N 个格子（网格）
3. 每下跌一格 → 买入一份；每上涨一格 → 卖出一份
4. 靠价格在区间内震荡反复赚取差价

**示意**：

```
20 元 ───── [卖出]
18 元 ───── [卖出]
16 元 ───── 持仓
14 元 ───── [买入]
12 元 ───── [买入]
10 元 ───── [买入]
```

**收益来源**：震荡，不是涨跌。

**风险**：
- 如果标的出现单边下跌且跌破下轨，持仓全部亏损
- 如果单边暴涨突破上轨，卖出后踏空

网格交易适合宽基 ETF、主流货币对，不适合个股（个股可能直接退市归零）。

## 8.7 如何选择适合自己的策略

没有最好的策略，只有最适合你的策略。选择时考虑以下维度：

**1. 时间投入**
- 几乎没时间 → 指数定投
- 每周花几小时 → 价值投资（选股后长持）
- 每天盯盘 → 趋势/短线，但请确保你有这个时间和纪律

**2. 心理素质**
- 持仓浮亏 30% 能坦然持有 → 可以做股票
- 浮亏 5% 就睡不着觉 → 从低风险品种开始，降低仓位

**3. 资金体量**
- 10 万以下：专注学习，不要分散在太多标的；指数投资是最优先级
- 100 万以上：可以构建多元化组合，考虑量化或多策略

**4. 能力圈**
- 你理解的行业，你更容易判断公司的好坏
- 程序员天然理解科技公司的商业模式，这是能力圈优势

**建议的起步路径**：
```
第一年：读书 + 看财报 + 模拟盘
第二年：小仓位实盘（总资产 10% 以内）
第三年：根据实盘复盘确定适合自己的策略，逐步加仓
```

## 本章小结

| 策略 | 核心逻辑 | 最适合 |
|---|---|---|
| 价值投资 | 低价买好公司，等市场认知修复 | 有耐心、能做企业分析的长期主义者 |
| 趋势交易 | 跟随趋势，严格止损 | 有纪律、能接受低胜率的交易者 |
| 量化交易 | 用模型替代人工判断 | 程序员，有数据能力 |
| 指数投资 | 定投宽基，分享市场整体增长 | 时间少、不想深入研究的大多数人 |
| 网格交易 | 震荡市场机械套利 | 判断标的长期区间震荡时 |
