# 第十一章：构建自己的交易体系

> 散户和职业交易者的核心差距不是信息量，而是有没有体系。没有体系的交易，是在用直觉赌博。

---

## 11.1 什么是交易体系：入场、出场、仓位的完整规则

**交易体系**是一套完整的、可重复执行的规则，回答三个问题：

1. **什么时候买**（入场条件）
2. **买多少**（仓位管理）
3. **什么时候卖**（出场条件，包括止盈和止损）

一个没有完整出场规则的"策略"不是策略，只是运气。

**体系的核心价值**：

- 消除情绪干预：规则触发则执行，不受今天新闻或账户浮亏影响
- 可以被评估和改进：有规则才能回溯，有回溯才能改进
- 保持一致性：一致性是判断策略有效性的前提

**简单体系示例（价值 + 趋势混合）**：

```
入场条件：
  - PE < 历史均值 20%
  - ROE > 15% 连续 3 年
  - 价格站上 MA60
  满足以上全部 → 考虑买入

仓位：
  - 单只股票不超过总资金 15%
  - 初始仓位 5%，确认上涨趋势后加至 15%

出场条件（止损）：
  - 价格跌破买入价 8% → 无条件止损
  - 价格跌破 MA60 → 减仓至 5%

出场条件（止盈）：
  - 达到合理估值区间（PE 接近历史均值） → 开始分批减仓
  - 基本面出现明显恶化（净利润连续 2 季下滑）→ 全部清仓
```

## 11.2 建立自己的选股池：行业 + 公司筛选框架

**不要试图覆盖所有股票**。聚焦在自己能理解的行业，建立一个 20–50 只股票的观察池，深度了解，持续跟踪。

### 行业筛选

优先关注：
- 你的职业相关行业（能力圈优势）
- 长期趋势向好的行业（老龄化 → 医药；数字化 → SaaS；绿色能源）
- 具有周期性但可以预判的行业

回避：
- 政策管控强、方向不清晰的行业（至少在看懂政策之前）
- 你完全不理解商业模式的行业

### 公司筛选（初步筛选条件参考）

用选股工具（东方财富选股器、雪球选股）设置基础财务过滤：

```
ROE > 12%（连续 3 年）
净利润增速 > 10%（近 3 年均值）
负债率 < 60%
市值 > 50 亿（流动性保证）
```

通过初筛的公司进一步阅读：
1. 最新年报（公司主营业务、风险因素、管理层讨论部分）
2. 近 8 个季度的利润表和现金流量表
3. 行业研究报告（券商研报作为参考，不作为决策依据）

## 11.3 买入信号与卖出信号的设计

信号设计的原则：**简单、明确、可量化**。复杂到自己都执行不了的信号，不如没有。

### 买入信号设计维度

| 维度 | 指标示例 |
|---|---|
| 估值 | PE < 历史 25 分位；PEG < 1 |
| 基本面触发 | 季报超预期；新产品发布；行业拐点确认 |
| 技术面确认 | 放量突破 MA60；MACD 金叉 |
| 情绪面 | 市场整体 VIX 高位；板块换手率极低 |

不需要所有维度都满足，但至少需要 2–3 个维度同时满足，避免单一信号误判。

### 卖出信号设计维度

**止损**（错了就走）：
- 技术位止损：跌破关键支撑，说明判断错误
- 固定比例止损：亏损超过 X%，无条件止损

**止盈**（赚了也要有纪律）：
- 达到目标估值：PE 接近历史均值上轨
- 基本面恶化：核心指标连续恶化
- 找到更好的机会：机会成本

**永远先确定卖出规则，再买入。**

## 11.4 回测的意义与局限：过去不等于未来

**回测（Backtesting）** 是用历史数据验证你的策略，看如果过去用这套规则交易，结果如何。

### 回测的价值
- 初步验证策略逻辑是否有基础
- 发现策略在不同市场环境（牛熊震荡）下的表现差异
- 估算大概的盈亏比和胜率

### 回测的陷阱

**过拟合（Overfitting）**：在历史数据上调参直到结果好看，但实际上是"记住了历史"而不是"发现了规律"。过拟合的策略在未来数据上几乎必然失败。

**幸存者偏差**：大多数数据库只包含现存的股票，已经退市的烂公司不在数据库里，导致回测结果虚高。

**未来函数**：回测代码中不小心用了当时尚未发布的数据（如用当天收盘价决策，但实际只能用开盘时的数据），造成虚假的高收益。

**交易成本低估**：没有真实模拟冲击成本（大单对价格的影响）和流动性限制。

**正确使用回测的姿势**：
1. 先有逻辑，再回测验证，不要靠回测"发现"规律
2. 留出样本外数据（如用 2010–2020 年数据训练，用 2021–2023 年数据验证）
3. 对结果保持怀疑，回测盈利不等于未来盈利

## 11.5 定期复盘：对的事情和错的事情各学什么

**复盘（Review）** 是提升最快的方法，但大多数人做得很差——只盯着"亏了多少"，不问"为什么亏"。

### 月度复盘框架

**本月结果**：
- 收益率 vs 沪深 300（基准）
- 最大回撤

**每笔交易复盘**：

```
买入理由是否被验证？
  → 是：好的判断，记录此类信号有效
  → 否：判断哪里出了问题？信息不全？逻辑缺陷？执行偏差？

出场是否按计划？
  → 是：纪律合格
  → 否：情绪因素（舍不得止损？追涨没按限价单？）
```

**季度复盘**：
- 整体胜率和盈亏比是否合理？
- 是否有某类交易持续亏损？（应该停止）
- 是否有某类交易持续盈利？（应该加大）

**年度复盘**：
- 长期跑赢基准了吗？（超额收益是否真实）
- 交易体系是否需要调整？
- 自己的能力圈有没有扩大？

## 11.6 持续学习的路径：从书本到实盘的闭环

学习股票是一个需要数年的过程，不要期望几个月速成。

### 学习路径建议

**第一阶段（1–3 个月）：建立认知框架**
- 读完本教程
- 读《聪明的投资者》（格雷厄姆）
- 读《彼得·林奇的成功投资》
- 开户，开始看财报（不买或极小仓位）

**第二阶段（3–12 个月）：精读财报，建立选股池**
- 精读 5–10 家公司的年报
- 建立自己的选股池和估值表格
- 小仓位实盘（总资产 5–10%），强制自己认真对待每笔决策
- 写交易日记，坚持复盘

**第三阶段（1–3 年）：形成体系，验证有效性**
- 经历至少一次完整的市场大波动（学到的最多）
- 评估自己的策略是否真的有效
- 确定自己的"能力圈"边界
- 根据实盘结果迭代体系

**始终保持的习惯**：
- 每季度读持仓公司的财报
- 每年读 1–2 本相关书籍
- 与有独立思考能力的人交流（而不是跟风追热点）

## 交易者的心智模型

几条值得反复思考的原则，来自经过时间检验的投资者：

> "价格是你付出的，价值是你得到的。" —— 巴菲特

> "市场短期是投票机，长期是称重机。" —— 本杰明·格雷厄姆

> "当别人贪婪时恐惧，当别人恐惧时贪婪。" —— 巴菲特

> "不要损失本金，不要忘记第一条。" —— 巴菲特

> "知道自己不知道什么，比知道很多更重要。" —— 查理·芒格

> "最好的投资，是投资自己的认知。"

## 本章小结

| 概念 | 核心要点 |
|---|---|
| 交易体系 | 入场 + 仓位 + 出场三要素缺一不可 |
| 选股池 | 聚焦能力圈，20–50 只深度跟踪，优于广撒网 |
| 信号设计 | 简单可量化，多维度确认，止损先于止盈 |
| 回测 | 验证逻辑的工具，不是优化参数的工具，警惕过拟合 |
| 复盘 | 每笔交易记录理由，定期评估决策质量而非结果 |
| 学习路径 | 书本 → 小仓位实盘 → 复盘迭代，需要 2–3 年建立体系 |
